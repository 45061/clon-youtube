## Objetivos

La aplicación le permite a los usuarios:

- Listar videos.
- Registrarse.
- Ingresar / Salir.
- Cambiar contraseña.
- Recuperar contraseña.
- Ver un video.
- Dejar un comentario en un video.
- Dar like a un video.
- Compartir un video.
- Embeber un video.
- Buscar videos.
- Subir un video.

Cada video debe tener almenos

- Título
- URL del video.
- Fecha de publicación.
- Descripción
- Etiquetas
- Número de likes
- Número de vistas

### Bonus

- Ver historial de videos.
- Crear listas de videos.
